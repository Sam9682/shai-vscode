Shai VS Code Extension - Setup Guide

Detailed instructions for setting up and configuring the Shai VS Code Extension

Prerequisites

  • Visual Studio Code version 1.74.0 or higher
  • Shai AI command-line tool installed and accessible in your PATH
  • Node.js (for building from source)

Installation Methods

Method 1: Install from .vsix file

Download the latest .vsix file and install it directly in VS Code:

code --install-extension shai-vscode-0.0.1.vsix

Method 2: Build from Source

  1. Clone or download the repository
  2. Navigate to the project directory
  3. Install dependencies: 
    npm install
  4. Compile the extension: 
    npm run compile
  5. Package the extension: 
    npx vsce package
  6. Install the generated .vsix file in VS Code

Configuration

Basic Configuration

Open VS Code Settings (Ctrl+,) and search for "Shai VS Code". Configure the following settings:

  • shai-vscode.shaiCommand: Path to the Shai AI command
    • Default: shai
    • Examples: /usr/local/bin/shai, python /path/to/shai.py, ./shai
  • shai-vscode.useWSL: Enable/disable WSL usage on Windows
    • Default: null (auto-detect)
    • Set to true to force WSL usage
    • Set to false to force native shell usage

Advanced Configuration

  • shai-vscode.useServer: Enable server mode for improved performance
    • When enabled, starts a long-running Shai server process
    • Provides real-time updates during processing
  • shai-vscode.serverUrl: URL of the local Shai HTTP/SSE server
    • Default: http://127.0.0.1:8000
    • Only used when useServer is true

Windows-Specific Setup

The extension automatically detects Windows and runs Shai commands through WSL bash, since Shai AI is only available for Linux/Mac.

Prerequisites for Windows:

  • WSL (Windows Subsystem for Linux) installed
  • Shai AI installed in your WSL environment

Verification Steps:

  1. Verify WSL is installed and working: wsl --version
  2. Test Shai in WSL: wsl bash -c "shai hello"
  3. Ensure Shai is installed in your WSL environment

Testing the Installation

  1. Restart VS Code
  2. Open the Shai panel from the Activity Bar (left sidebar)
  3. Enter a simple command like "hello" in the chat input
  4. Press Enter or click Send
  5. Verify that Shai responds appropriately

Troubleshooting

Common Issues and Solutions

  • Command not found: Ensure Shai is installed and accessible in your PATH
  • WSL not detected: Verify WSL is installed and working with wsl --version
  • No output from commands: Test the Shai command directly in your terminal

Debugging Tips

  • Check the VS Code Developer Tools Console for errors
  • Verify the Shai command works outside of VS Code
  • Confirm the correct path is set in extension settings
