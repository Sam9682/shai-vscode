OPC-Wiki: Outdated and Incorrect Documentation

Note: This page documents known issues, outdated information, and incorrect documentation found in the project documentation.

Known Issues in Documentation

1. Windows Support Documentation

The documentation states that the extension automatically detects Windows and runs commands through WSL, but this may not work correctly in all environments.

Issue: The WSL detection logic may fail on some Windows systems where WSL is installed but not properly configured.

2. Installation Instructions

The installation instructions mention installing from the .vsix file, but newer versions may require different installation methods.

Issue: The .vsix file may not be compatible with newer VS Code versions or may require additional steps for installation.

3. Prerequisites Section

The prerequisites section mentions Shai AI command-line tool but doesn't specify required versions or compatibility.

Issue: Missing version requirements for Shai AI tool which could cause compatibility issues.

If you find outdated or incorrect information in the documentation, please:
